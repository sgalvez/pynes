# v0.0.4 - Render scrolled backgrounds and smooth audio

Release date: 2026-07-01

Tag: `v0.0.4`

Tag commit: `e674730`

Compare: `v0.0.3...v0.0.4`

## Summary

This release improves background rendering for games that scroll or switch
nametables. It also experiments with audio smoothing to reduce harshness in the
early APU output.

## Added

- Scroll-aware background rendering using `PPUSCROLL`.
- PPUCTRL base-nametable selection for background rendering.
- Cartridge mirroring awareness for nametable reads.
- Background dirty tracking for scroll, nametable, palette, and CHR changes.
- Audio smoothing in the APU output path.
- Tests for scrolled background rendering and cache invalidation behavior.

## Changed

- Background rendering now samples a larger tile grid and crops it through the
  current scroll offset.
- PPU rendering better follows horizontal/vertical nametable mirroring.

## Fixed

- Background layers now move with scroll values instead of remaining fixed.
- More games show the expected background region after scroll writes.

## Validation

- `python -m pytest`
- `python -m ruff check .`
- release workflow package and standalone artifact checks

## Known Limitations

- Full PPU scroll timing is still simplified.
- Audio smoothing in this release can make playback sound dull or incorrect for
  some ROMs.
- Screen transitions that depend on more accurate PPU timing may still be wrong.
